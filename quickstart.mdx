# Quickstart

### Step 1: Get Your API Key

<AccordionGroup>
  <Accordion icon="key" title="Obtain API Key">
    Contact your administrator or sales representative to get your API key.
  </Accordion>
  <Accordion icon="terminal" title="Set Up Environment">
    1. Install your preferred SDK (OpenAI, Anthropic, etc.)
    2. Set your API key
    3. Configure the gateway URL
  </Accordion>
</AccordionGroup>

### Step 2: Modify Your Code (2 Steps!)

<AccordionGroup>
  <Accordion icon="code" title="Update SDK Configuration">
    **For Anthropic SDK:**
    ```python
    from anthropic import Anthropic
    client = Anthropic(
        api_key="your-pipellm-key",
        base_url="https://api.pipellm.com/v1"  # Point to our gateway
    )
    ```

    **For OpenAI SDK:**
    ```python
    import openai
    client = openai.OpenAI(
        api_key="your-pipellm-key",
        base_url="https://api.pipellm.com/v1"  # Point to our gateway
    )
    ```
  </Accordion>

  <Accordion icon="play" title="Test Your Connection">
    ```python
    response = client.messages.create(
        model="claude-3-sonnet",
        messages=[{"role": "user", "content": "Hello"}]
    )
    print(response.content[0].text)
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Go Live

<Accordion icon="rocket" title="Deploy">
  1. Update your production configuration
  2. Monitor usage via management dashboard
  3. Start enjoying better performance and cost optimization!
</Accordion>
