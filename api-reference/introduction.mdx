# API Reference

PipeLLM Gateway supports multiple API formats. Use your familiar native SDK - just change the baseURL.

## Quick Start

Choose your API format:

| API Format | Supported Models | Endpoint |
|------------|------------------|----------|
| [OpenAI Compatible](/api-reference/openai/chat-completions) | GPT, Grok, DeepSeek, Groq, etc. | `/v1/chat/completions` |
| [Anthropic](/api-reference/anthropic/messages) | Claude series | `/v1/messages` |
| [Gemini](/api-reference/gemini/generate-content) | Gemini series | `/v1beta/models/{model}:generateContent` |

## Authentication

All requests use your PipeLLM API Key with format-specific headers:

| API Format | Header |
|------------|--------|
| OpenAI Compatible | `Authorization: Bearer $PIPELLM_API_KEY` |
| Anthropic | `x-api-key: $PIPELLM_API_KEY` |
| Gemini | `x-goog-api-key: $PIPELLM_API_KEY` |

## Base URL

```
https://api.pipellm.com
```
