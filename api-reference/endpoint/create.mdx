---
title: 'Text Completions'
openapi: 'POST /v1/completions'
---

<Note>
  This endpoint is for text completion, similar to the legacy OpenAI completion API.
</Note>

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| model | string | ✅ | Model name (e.g., "gpt-3.5-turbo") |
| prompt | string | ✅ | Input text |
| max_tokens | number | ❌ | Maximum tokens to generate |
| temperature | number | ❌ | Sampling temperature (0-2) |

## Example

```json
{
  "model": "gpt-3.5-turbo",
  "prompt": "从前有一只",
  "max_tokens": 100,
  "temperature": 0.8
}
```

## Response

```json
{
  "id": "comp-123",
  "choices": [
    {
      "text": "从前有一只聪明的狐狸，在森林里过着快乐的生活。",
      "index": 0,
      "finish_reason": "length"
    }
  ],
  "usage": {
    "prompt_tokens": 5,
    "completion_tokens": 20,
    "total_tokens": 25
  }
}
```

## Use Cases

- Text generation
- Story continuation
- Code completion
- Creative writing
