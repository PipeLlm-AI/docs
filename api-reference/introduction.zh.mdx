# API 参考

PipeLLM Gateway 支持多种 API 格式。使用您熟悉的原生 SDK，只需更改 baseURL。

## 快速开始

选择您使用的 API 格式：

| API 格式 | 支持的模型 | 端点 |
|----------|------------|------|
| [OpenAI 兼容](/api-reference/openai/chat-completions) | GPT、Grok、DeepSeek、Groq 等 | `/v1/chat/completions` |
| [Anthropic](/api-reference/anthropic/messages) | Claude 系列 | `/v1/messages` |
| [Gemini](/api-reference/gemini/generate-content) | Gemini 系列 | `/v1beta/models/{model}:generateContent` |

## 认证

所有请求使用 PipeLLM API Key，不同格式使用不同的 Header：

| API 格式 | Header |
|----------|--------|
| OpenAI 兼容 | `Authorization: Bearer $PIPELLM_API_KEY` |
| Anthropic | `x-api-key: $PIPELLM_API_KEY` |
| Gemini | `x-goog-api-key: $PIPELLM_API_KEY` |

## Base URL

```
https://api.pipellm.com
```
