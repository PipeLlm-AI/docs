# Development

Set up your PipeLLM Gateway development environment.

## ðŸš€ Quick Setup

<Steps>
<Step title="Install SDKs">
Install your preferred LLM SDKs:

```bash
# For OpenAI
pip install openai

# For Anthropic
pip install anthropic

# For Google
pip install google-generativeai
```
</Step>

<Step title="Configure Environment">
Set up your environment variables:

```bash
export PIPELLM_API_KEY="your-pipellm-key"
export PIPELLM_BASE_URL="https://api.pipellm.com/v1"
```
</Step>

<Step title="Test Integration">
Test your setup:

```python
import os
from anthropic import Anthropic

client = Anthropic(
    api_key=os.getenv("PIPELLM_API_KEY"),
    base_url=os.getenv("PIPELLM_BASE_URL")
)

response = client.messages.create(
    model="claude-3-sonnet",
    messages=[{"role": "user", "content": "Hello"}]
)
print(response.content[0].text)
```
</Step>
</Steps>
